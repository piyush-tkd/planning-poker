Resetting Test Data

To reset all test data and start fresh:

  1. Run database reset script: npm run reset:db
  2. Re-seed with test data: npm run seed:test
  3. Verify super admin account exists: bahetipiyush@gmail.com
  4. Login via Google OAuth and verify admin access
